# Монетизация и экономика Syncwatch

## Принцип: убыток невозможен, если не покупать подпись раньше времени

Все инфраструктурные расходы = 0 (см. таблицу). Единственный расход — подпись кода,
и он опционален. Не покупать, пока доход не оправдает.

| Статья | Стоимость |
|---|---|
| Сигналка (Cloudflare Workers free) | 0 (100k req/день) |
| TURN (Cloudflare Realtime free) | 0 (только control-трафик, ~КБ; 1 ТБ/мес не выбрать) |
| Страница загрузки | 0 (статика на AnimeDB + GitHub Releases) |
| Домен | 0 (AnimeDB уже есть) |
| Подпись кода | 0 сейчас / ~$120/год позже (Azure Trusted Signing ~$10/мес) |

## Источники дохода (по убыванию реального потенциала)

1. **Воронка в AnimeDB (главный, косвенный).** Каждая co-watch сессия возвращает людей
   на сайт выбирать контент → рост трафика/показов, которые сайт уже монетизирует.
   Постоянный поток, а не разовый донат. Приложение = retention-машина сайта.
2. **Донаты Boosty + Tribute (прямой, бесплатный).** Конверсия 0.5–2% активных.
   Tribute = иностранцы + крипта, Boosty = подписки + разовые. Текст: «проект существует
   только благодаря вашим пожертвованиям».
3. **Freemium supporter-перки (позже).** Косметика: темы, комнаты >N чел, кастомный код
   комнаты, цвет ника. Выше конверсия, чем донат. Добавляет платёжную поверхность и сложность.
4. **Реклама в приложении — НЕ делать.** Скам-вид + усиливает антивирус-эвристику
   (сетевые вызовы к ad-сетям). Реклама допустима только на сайте/странице загрузки.

## Антивирусы бесплатно

**Не триггерить эвристику:**
- Без обфускации, без упаковки (никакого UPX), без записи в автозапуск реестра,
  без инъекций в чужие процессы, без вызовов рекламных сетей.
- Апдейтер — главный риск (качает+запускает); смягчено подписью minisign + скачиванием
  только с GitHub Releases.
- Спавн mpv/VLC — легитимные плееры, документировать в описании.

**Бесплатные white-list (после первого билда):**
Kaspersky Allowlist, Microsoft WDSI, Dr.Web, VirusTotal FP-репорты. Репутация SmartScreen
набирается сама от объёма скачиваний с GitHub. См. [RELEASE.md](RELEASE.md).

**Платно только при доходе:** Azure Trusted Signing (~$10/мес) — дешевле EV-сертификата,
полностью убирает SmartScreen.

## Порог решений

- Подпись кода покупать, когда донаты/трафик стабильно > стоимости подписи.
- Freemium вводить, когда есть база активных комнат (иначе некому платить).
